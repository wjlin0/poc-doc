# 章管家前台任意文件上传漏洞(XVE-2024-19042)

章管家是上海建业信息科技股份有限公司推出的一款针对传统印章风险管理提供的整套解决方案的工具。 该系统存在前台任意文件上传漏洞，攻击者成功利用该漏洞可以上传任意文件，从而获得服务器权限。

## fofa

```yaml
app="章管家-印章智慧管理平台"
```
